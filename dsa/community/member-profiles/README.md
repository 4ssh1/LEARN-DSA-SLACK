# Community Member Profiles

Welcome to our member directory! Meet the learners and mentors in our DSA community.

## How to Add Your Profile

1. Create a copy of the [profile template](template.md)
2. Fill in your information
3. Save as `your-github-username.md` in this directory
4. Submit a pull request
